<h1 style="text-align:center;">ANIDL-GUI</h1>

ANIDL-GUI is a simple and efficient anime downloader that allows you to download your favorite anime series with just a few clicks.

## Features

- Download anime series in high quality
- Download multiple episodes at once
- Download anime in full HD quality
- Download anime in 1080p quality
- Always downloads 4 episodes per anime simultaneously

## Installation
If you are on windows, it is recomended to download the .msi file because if you don't you might not be able to update the app.
[Release](https://github.com/zerodev-exe/anidl-gui/releases/latest/)
